Multi-Language Text Summarizer

View on GitHub

Overview

A Python-based tool that efficiently summarizes text in multiple languages, including English, Marathi, and Mixed-Language (English, Marathi, and Hindi). It uses natural language processing (NLP) to provide concise summaries and offers various reduction modes for customizable summary lengths.

Core Features

  • Multi-Language Support: Summarizes text in English, Marathi, and mixed-language mode (English, Marathi, and Hindi).
  • Reduction Modes: Allows users to choose between Less and More Reduction for customizable summary lengths.
  • Interactive UI: Features a simple, user-friendly interface built with Jupyter Notebook widgets for easy text input and summary generation.
  • Real-Time Processing: Summarizes text in real-time, providing instant results.
  • Custom Stopwords Handling: Implements language-specific stopword lists to improve summary accuracy.

Technologies Used

  • Python: Primary language for developing the summarizer.
  • NLTK:Used for tokenization, sentence segmentation, and stopword handling.
  • ipywidgets: Provides interactive UI components for the Jupyter Notebook environment.
  • Google Colab/Jupyter Notebook: The environment used to run and display the summarizer.

Challenges and Solutions

  • Multi-Language Support: Handled multi-language support and mixed-language texts by creating custom stopword lists and ensuring compatibility across languages
  • Mixed-Language Texts: Supporting a mix of languages required a flexible stopword system, it was achieved by merging stopword lists for the mixed-language mode.

Impact and Applications

Ideal for educational use, content curation, and research, allowing users to extract key points from text in multiple languages. The tool efficiently summarizes large volumes of text for multilingual audiences without switching tools.

Conclusion

A versatile, multi-lingual text summarizer that offers customizable summaries and an interactive UI. It addresses the challenges of supporting multiple languages and mixed-language texts, making it suitable for educational and research purposes.
